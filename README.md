# SkinGPT Integration Examples

Example apps demonstrating SkinGPT integration for various platforms. Feel free to reach out if you run into any issues or if your integration is missing.

## Instructions

1. Choose a platform folder
2. Open its README.md file
3. Follow the instructions there

## Available Examples

- **html-js/** - Pure HTML/CSS/JavaScript integration (zero dependencies)
- **react-ts/** - React + TypeScript integration with Vite

Explore and enjoy!
